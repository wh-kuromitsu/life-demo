# にじいろ統合管理システム — モック v2.0

## 概要

株式会社エデュケーションNET（にじいろグループ、13事業所×2法人）向け統合管理システムの提案モック。
補助金採択案件（新事業進出補助金）の要件定義v2.0に基づく全面刷新版。

## デザイン方針

**「精緻な業務設計（Editorial Operational）」**
- 温かみのあるクリーム基調 + テラコッタ（強調）+ セージグリーン（完了）
- Bricolage Grotesque（ディスプレイ）+ Zen Kaku Gothic New（本文）+ JetBrains Mono（数値）
- 業務ソフトとしての静謐・濃密・正確な情報表現

## セットアップ

```bash
npm install
npm run dev
```

→ `http://localhost:5173/` で開きます。

## 画面構成（11ページ）

### オペレーション
1. **ダッシュボード** - 本日の全体俯瞰、KPI、施設別利用者数
2. **⚡ ルート最適化** - フラグシップ画面（詳細は下記）
3. **送迎管理** - 便制カレンダー（曜日×方向）、本日の便詳細
4. **入退室記録** - 打刻、月間出席カレンダー

### 児童・支援
5. **児童管理** - 30名サンプル、50項目リッチ表示、タグ絞り込み
6. **個別支援計画** - 6ステップワークフロー、5領域別目標設定

### 拠点・備品
7. **施設管理** - 13事業所×2法人、特化テーマ、補助金対象
8. **備品・教具** - 28種サンプル、5領域×年齢帯検索

### システム
9. **HUG連携** - 同期ステータス、差分、ログ
10. **通知**
11. **設定** - 法人/車両/通知/アカウント/セキュリティ

## フラグシップ：ルート最適化

3ステップ構成：
1. **条件設定** - 制約チップ7種（支援時間1h35m/18:30退勤/国道NG/同乗NG/性別配慮）、除外児童
2. **シミュレーション** - Pattern A (OR-Tools VRP) / Pattern B (K-Means) 並列比較
3. **確定・ナビ連携** - Google Maps URL生成、ドライバー配布PDF

### 特徴
- **SVG春日井市マップ** - 学校/学童/施設/ルートを可視化、国道を制約線として表示
- **未配置者パネル** - 松浦先生の認知モデル継承（学校ごと時間順、11分差で便分け）
- **実運用パターン織込** - 春日井翔陽連携/祖父母宅/学童経由/時間指定/同乗NG/イレギュラー
